Turn Rmarkdown output into Github Pages

If you Google this topic you will find this GitHub and RStudio tutorial, which is quite long.

Here is the short version:

Name the output html file as index.html and go to repo settings -> GitHub Pages and point the source to where this file is. Also you can change theme just by clicking the buttom.

You can assign the short domain names like liuyanguu.github.io to only one repo (in my case, this blog). All the rest will have to add the repo name behind: like liuyanguu.github.io/COVerAge_Plot
